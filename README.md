# ✈️ RoamSmart AI: Intelligent Student Travel Planner

An advanced, data-driven budget optimization and predictive itinerary engine designed explicitly for students looking to maximize travel experiences under strict financial constraints. Built as a capstone project for the **IBM & Edunet AI Internship Track**.

## 🚀 Key Architectural Features
- **Automated Failover Core:** Features built-in structural exception handling to counter live network spikes and public server bottlenecks (`503 Errors`) gracefully.
- **Dynamic Optimization Matrix:** Processes inputs across variable destination nodes, precise currency layers, and strict financial constraint tiers.
- **Integrated Data Visualizations:** Displays data allocation analytics using native interactive bar charts mapping travel cost segments.
- **Standalone Delivery:** Hardcoded secure token routing removes any friction for end-users, delivering a true single-click compilation experience.

## 🛠️ Technical Stack
- **User Interface:** Streamlit Framework
- **Model Engine:** Google Gemini 2.5 Flash (`gemini-2.5-flash`)
- **SDK Integration:** Modern `google-genai` Production Wrapper
- **Data Structuring:** Pandas Core DataFrames
